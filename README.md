## ðŸ“¦ Installation

1. Copy `horizon-grove.toml` into your Helix themes directory:

   ```sh
   mkdir -p ~/.config/helix/themes
   cp horizon-grove.toml ~/.config/helix/themes/
   ```

2. Edit your `~/.config/helix/config.toml` and set:

   ```toml
   theme = "horizon-grove"
   ```
